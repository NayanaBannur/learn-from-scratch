In the early internet, that directory was a single text file called `HOSTS.TXT`. It listed every host name and its address, one per line. [[cite:rfc1034]]

One organisation maintained the master copy, and every machine downloaded the whole file to stay current. It worked while the network was small.

As hosts multiplied in the early 1980s, this broke down. The file grew huge, every change meant redistributing it everywhere, and two people could not add a name without colliding. A single central list could not scale.
