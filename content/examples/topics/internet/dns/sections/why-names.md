Every machine on the internet is reached by an **IP address** — a number like `93.184.216.34`. Routers use these numbers to move traffic, but people don't remember them.

DNS, the **Domain Name System**, is the internet's directory. It turns a name a person can remember, like `example.com`, into the IP address a machine needs.

So when you open a site, a lookup happens first: your device asks DNS for the name, gets back an address, and only then connects. Almost every connection you make starts with this translation.
